## Support Process

```
1. Emergency reported
   → Channel (phone, chat, app)
   → Basic information collected

2. Case created
   → Category assigned
   → Severity assessed
   → Case routed

3. Response initiated
   → Assigned to specialist
   → Partner contacted if needed
   → Traveler updated

4. Case managed
   → Actions taken
   → Status updates
   → Resolution pursued

5. Case closed
   → Resolution documented
   → Follow-up scheduled
   → Feedback collected
```
