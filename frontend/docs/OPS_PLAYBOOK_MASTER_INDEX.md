# Agent Operations Playbook — Master Index

> Research on travel agent daily operations, standard operating procedures, escalation protocols, quality metrics, and operational workflows for the Waypoint OS platform.

---

## Series Overview

This series covers the operational side of running a travel agency — from agent daily routines and structured SOPs to escalation matrices and quality assurance. While sales playbooks cover how to sell, operations playbooks cover how to execute consistently. The difference between agencies that scale and those that plateau is operational discipline — every inquiry gets the same quality response, every booking follows the same verification process, every disruption has a pre-defined response.

**Target Audience:** Operations managers, agency owners, agent trainers

**Key Insight:** The best agencies have 90%+ SOP adherence because their systems enforce it — not because agents memorize procedures. A proposal can't be sent without a margin check. A booking can't be confirmed without advance payment. A pre-departure check can't be skipped because the system sends it automatically. System-enforced SOPs are more reliable than training-enforced SOPs.

## Key Themes

### 1. Structure the Day
A productive agent doesn't react to whatever comes in — they follow a structured day: morning review, active selling in the morning, operations in the afternoon, wrap-up before leaving. Structure ensures nothing falls through the cracks.

### 2. SOPs as System Rules
Every SOP should be enforced by the platform where possible. If the SOP says "margin check before sending proposal," the system should require it. If it says "advance payment before confirming booking," the system should block confirmation until payment is recorded.

### 3. Escalate Exceptions, Not Routine
Agents should handle 90% of situations autonomously using SOPs. Only 10% — financial exceptions, supplier failures, safety issues — need escalation. Clear escalation boundaries prevent both under-escalation (problems hiding) and over-escalation (owner bottleneck).

### 4. Measure What Matters
Track response time, proposal turnaround, booking accuracy, and NPS per agent. These four metrics predict agent performance better than revenue (which is influenced by lead quality and seasonality).
