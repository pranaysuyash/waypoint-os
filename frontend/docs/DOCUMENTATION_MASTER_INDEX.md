## Document Format

All research documents follow a consistent format:

1. **Key Questions** — What we need to learn
2. **Research Areas** — Organized exploration topics
3. **Data Model Sketch** — TypeScript interfaces for structure
4. **Open Problems** — Known challenges and options
5. **Next Steps** — Suggested research activities

This format emphasizes **research and exploration** over premature implementation.
