# Destination Event & Festival Intelligence — Master Index

> Research on event databases, demand prediction, event package design, and event analytics for travel agencies.

---

## Series Overview

This series covers how events, festivals, sports, and concerts drive travel demand — from building and maintaining an event database to creating event-specific packages, managing event inventory, and measuring event performance.

**Target Audience:** Product managers, revenue managers, marketing teams, agency owners

**Key Constraint:** Event dates shift (lunar calendars), inventory is scarce and perishable, and pricing during events requires careful balance between premium and customer goodwill.
