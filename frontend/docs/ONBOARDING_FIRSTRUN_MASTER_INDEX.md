## Key Themes

- **Time-to-first-value is critical** — If an agency can't create their first trip within 2 hours of signup, they'll leave. Setup wizard must be fast with smart defaults.
- **Progressive disclosure** — Show core features first, unlock advanced features as the agent gains experience. Don't overwhelm on day one.
- **Onboarding tracks** — A new travel agent and an experienced agent switching platforms need different onboarding. One flow doesn't fit all.
- **Help is contextual** — The best help appears exactly when needed, not buried in a knowledge base. Tooltips, inline tips, and error-linked articles reduce support load.
- **Activation is measurable** — Activation isn't "they signed up." It's "they completed a booking and generated revenue." The scoring model quantifies this.

## Integration Points

- **Agent Training** — Onboarding is the first phase of agent training; they connect seamlessly
- **Gamification** — Activation events and achievements feed into the gamification system
- **Task Management** — Onboarding generates tasks (setup steps, training modules)
- **Template System** — Quick-start templates reduce first-trip friction
- **Notification System** — Onboarding nudges and churn interventions use notifications
- **Analytics & BI** — Activation metrics and engagement data feed into analytics
