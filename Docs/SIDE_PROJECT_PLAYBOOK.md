# Waypoint OS as a Side Project

**Context**: This project is not a VC-backed startup. It's a side project for a solo founder.
**Date**: 2026-04-30

---

## 1. Mindset Shift: Side Project vs Startup

| Dimension | Startup Mode | Side Project Mode |
|-----------|-------------|-------------------|
| Growth goal | 100%+ YoY | 10-20% month over month is great |
| Time investment | Full-time + overtime | Evenings + weekends, ~10-15 hrs/week |
| Revenue target | $1M+ ARR or bust | ₹20-50K/mo extra income is a win |
| Hiring | Hire fast | Don't hire. Automate. Stay solo. |
| Fundraising | Required for growth | Not needed. Revenue funds everything. |
| Risk tolerance | High (investor money) | Low (your own time and money) |
| Success metric | Exit / massive scale | Profitability + paying users + learning |

## 2. What Changes for Waypoint OS

### Growth Expectations

Founder-led at 5 new customers/month is actually aggressive for a side project.

**More realistic side project pace**:

| Metric | Startup Assumption | Side Project Reality |
|--------|-------------------|---------------------|
| New customers/month | 5-10 | 2-4 (evenings + weekends outreach) |
| Time to 50 customers | 12 months | 18-24 months |
| Churn | 3-5% target | 5-7% (less time for support) |
| Add-ons built | 3 in first year | 1 in first year (flight tracking) |
| LLM cost optimization | Important | Critical — keep it under ₹500/mo total |

### Cost Model (Side Project)

```
Fixed monthly: ₹5,000-8,000 (not ₹15,000)
  - Hosting: ₹3,000 (Vercel free tier + cheap DB)
  - LLM base: ₹1,000-2,000 (minimal usage)
  - Tools: ₹1,000-3,000 (free tiers wherever possible)

Variable per customer: ₹400 (same — LLM + infra + support)

One-time: ₹50,000 (not ₹1,00,000)
  - Deploy: ₹20,000
  - Legal: ₹20,000 (basic docs)

```

## 4. Side Project Playbook

### Months 1-3: Launch + First 5 Customers

```
Week 1-2: Ship working MVP (already done!)
Week 3-4: Offer free to 3 agencies you know
           Goal: process 5 real inquiries each

Month 2: Convert 2 to paying ($20/mo)
         Post in 1 Facebook group, answer questions

Month 3: Get to 5 paying customers
         Start 1 referral conversation
         ₹9,300/mo MRR
```

### Months 4-9: Build Momentum

```
Goal: 10-15 customers
       ₹18,600-27,900/mo MRR

Key activities:
  - 2 Facebook posts/week
  - 1 WhatsApp community engagement/week
  - Check in with every customer monthly
  - Build 1 small feature based on customer requests
  - Start annual billing push (20% off)

Milestone: ₹20K/mo covers all costs + some profit
```

### Months 10-18: Stabilize

```
Goal: 20-30 customers
       ₹37,200-55,800/mo MRR

Key activities:
  - Refine onboarding (reduce support time)
  - Automate billing, reporting
  - Build 1 add-on (flight tracking) — offer it to existing customers
  - Start referral program (1 month free for both sides)

Milestone: ₹50K/mo = healthy side income
```

### Year 2-3: Decide

```
At 40-60 customers:
  Option A: Keep as side project. ₹75K-1.1L/mo passive-ish income.
  Option B: Go full-time if growth accelerates (200+ customers).
  Option C: Sell the business (micro-SaaS acquisition).
```

## 5. Anti-Patterns for Side Projects

| Anti-Pattern | What It Looks Like | Prevention |
|-------------|-------------------|------------|
| **Shiny feature syndrome** | "I need to build flight tracking, price intel, WhatsApp API, and a mobile app" | Pick ONE feature. Build it. Ship it. Move on. |
| **Over-engineering** | "I need microservices, Redis, and a CDN for 20 customers" | One $5 VPS or Vercel free tier. Done. |
| **Support overload** | Answering 20 emails/day for 30 customers | Build FAQ, better onboarding, limit support channels |
| **Guilt about slow growth** | "I should be growing faster" | You're one person with a side project. Slow is sustainable. |
| **Pricing too low from fear** | "₹999 is cheap, they'll definitely buy" | $20/mo for a tool that saves 5+ hours/week is a steal. Don't discount. |

## 6. When to Go Full-Time

Signals that it's time to quit your day job:

```
Hard signals (need 2 of 3):
  - ₹1.5L+/mo MRR (₹18L/yr)
  - Growing 15%+ month over month without your active outreach
  - Support takes < 5 hrs/week because product is stable

Soft signals:
  - Customers are finding you (not the other way around)
  - You're turning down feature requests because you can't build fast enough
  - The side project income exceeds your day job income
```

Until those signals hit, keep it as a side project. The business model works. It'll grow on its own time.
