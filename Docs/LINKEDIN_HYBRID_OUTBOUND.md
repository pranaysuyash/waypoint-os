# LinkedIn Outreach — Hybrid Approach for Waypoint OS

**Context**: Full-time on this. Not a side project time budget. Not raising.
**Search results**: 10+ pages of "travel industry agency owner" on LinkedIn — plenty of targets.

---

## 1. The Problem with Pure LinkedIn DMs

LinkedIn DMs alone for a $20/mo product don't work well:

```
Manual DM → reply → chat → qualify → demo → trial → paid
Average time per conversion: 2-3 hours spread over 2 weeks
At 50% of your time = maybe 15-20 conversations/month
Close rate: ~15% = 2-3 new customers/month
That's ₹3.7-5.6K MRR for 80 hours of messaging
```

The bottleneck is the **DM conversation**. Every back-and-forth takes time and you can only do a few at once.

## 2. The Hybrid: LinkedIn + Demo-Driven

The fix: use LinkedIn only for the **first connection** and a **single low-friction CTA** — a 5-minute live demo where you process one of THEIR real inquiries.

```
LinkedIn: Connect → DM (1 message) → "Want to see it in 5 min?" → Calendar link
                                                                         ↓
                                                             5-min Zoom/Google Meet
                                                             Process their real inquiry
                                                                         ↓
                                                              Set up trial account
                                                                         ↓
                                                             14-day trial → paid
```

### Why this works for $20/mo

| Stage | Time per prospect | Who does it |
|-------|-------------------|-------------|
| Search & connect (LinkedIn) | 2 min | You |
| DM (1 message, personalized) | 3 min | You |
| 5-min demo (live, on calendar) | 5 min | You |
| Trial setup | 5 min | You (automated after demo) |
| Check-in during trial (3 emails) | 5 min total | Automated |
| Convert to paid | Automatic | Stripe |
| **Total per customer** | **~20 min** | (not counting no-shows) |

### The conversion funnel

```
100 connection requests sent
  → 40 accept (40%)
    → 15 reply to DM (37%)
      → 8 book a 5-min demo (53%)
        → 6 show up (75%)
          → 4 convert to trial (67%)
            → 2 convert to paid (50%)

From 100 connections → 2 paying customers
Time invested: ~4 hours for the outreach, ~1 hour for demos
= 5 hours for 2 customers = ₹3,720 MRR ($40)
```

At 2 customers per 100 connections, and 10+ pages of results available, you can run this repeatedly.

## 3. The LinkedIn to Demo Flow

### Step 1: Search & Filter (15 min/day)

Search queries:
1. `"travel agency" founder`
2. `"boutique travel agency" owner`
3. `"travel agency" director` (larger agencies)
4. `"travel agency" CEO`
5. `"travel agency" partner India`

Send 10-15 connection requests per day.

Connection note template:

```
Hi [Name] — noticed you run [agency name]. I'm working on a tool
that helps travel agencies cut inquiry processing time by 60%.
Would love to connect and hear your take on the space.
```

### Step 2: DM After They Accept (same day)

```
[Name], thanks for connecting.

Quick question — how do you currently handle new trip inquiries
coming in? Specifically from WhatsApp or email to first quote.

I'm not selling anything — just trying to understand the real
process on the ground. Would genuinely value your take.
```

### Step 3: Qualification (their answer tells you)

| Their Answer | What It Means | Next Step |
|-------------|---------------|-----------|
| "We use spreadsheets, takes forever" | **Hot lead** — feeling the pain | "Want to see something I built that handles this in 30 seconds?" |
| "We have a system that works okay" | Warm — might convert with demo | "Curious — what's the most frustrating part of that system?" |
| "We use [specific tool]" | Warm — need to differentiate | "How long did it take to set up? Did it actually save time?" |
| "We're too small for tools" | Cold — not ready | "Totally understand. If you ever grow to 10+ inquiries/week, this might help." |

### Step 4: The 5-Minute Demo (The Closer)

This is your killer conversion tool. It's not a feature walkthrough. It's:

```
1. "Paste one real inquiry you received this week." (30 sec)
2. [Process it in Waypoint OS while sharing screen] (60 sec)
3. "Here's what it extracted. Accurate? Anything it missed?" (60 sec)
4. "See the gap analysis it flagged?" (30 sec)
5. "Here's the strategy bundle it generated." (30 sec)
6. "Want me to set up your account? Takes 2 minutes." (60 sec)
```

**Total**: ~5 minutes. Not 30. Not an hour.

### Step 5: Trial Setup (during the call)

Don't send them to sign up themselves. Set it up FOR them:

```
"Check your email — I just created your account. You're on a 14-day free trial.
No credit card needed. This week, try processing 3 real inquiries.
I'll check in on day 7 to see how it's going."
```

### Step 6: Automated Trial Nurture

```
Day 1 (auto): "Your trial is active. Here's a 2-min video showing the flow."
Day 3 (auto): "Have you processed an inquiry yet? Reply if you need help."
Day 7 (you): "How's it going? Any questions?" [personal check-in]
Day 12 (auto): "Your trial ends in 2 days. Want to continue? Click here to subscribe."
Day 14 (auto): "Your trial has ended. Your data is safe for 30 days."
```

## 4. Time Budget (Full-Time, 40 hrs/week)

You have full-time to invest. Here's how to spend it:

```
OUTREACH (20 hrs/week)
  LinkedIn connection requests + DMs: 2 hrs/day (10 hrs/week)
  Facebook groups (value posts + replies): 2 hrs/day (10 hrs/week)

DEMOS (10 hrs/week)
  5-min demos: ~5-10 demos/week (5-10 hrs)
  Buffer for reschedules + no-shows

PRODUCT (8 hrs/week)
  1-2 days per week building features, fixing bugs
  Based on what demo conversations reveal

CONTENT (2 hrs/week)
  Convert 1 doc into a blog post or LinkedIn post
  Repurpose existing research

TOTAL: 40 hrs/week
```

At this pace: **8-20 demos/week → 2-5 new customers/week → 8-20 new customers/month**.

That's **50-100 customers in 3-6 months** at ₹3,000 avg ARPU = ₹1.5-3L MRR.

## 5. What You Need Before Doing This

| Requirement | Status | Action |
|-------------|--------|--------|
| Working demo that can process a real inquiry | ✅ Likely ready | Test with your own inquiry first |
| Calendar link for booking demos | ❌ | Set up Calendly or similar |
| Screen recording capability | ✅ Built into Zoom/Meet | QuickTime or Loom for async |
| Trial account auto-provisioning | ❌ | Need to set up so you can create accounts during the call |
| Stripe billing for trial→paid | ❌ | Need to connect payments |
| 2-min demo video (for async) | ❌ | Record a Loom walking through one inquiry |

### Minimum viable setup (can do in a day)

```
1. Calendly → "5-min Waypoint OS demo" scheduling link
2. Ability to create accounts manually during the call
3. A single real inquiry to demo with (yours or a friend's)
4. Stripe test mode connected for trial→paid
```
