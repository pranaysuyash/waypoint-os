# Review Assets

This folder stores screenshot evidence used by review/status docs in `Docs/review/`.

## Naming convention

Use descriptive, dated filenames:

- `<feature-or-scope>-<variant>-YYYY-MM-DD.png`

Examples:

- `overview-action-required-grouped-2026-05-19.png`
- `overview-action-required-grouped-mobile-2026-05-19.png`

## Guidance

- Keep both desktop and mobile captures when relevant.
- Prefer PNG for UI fidelity.
- Store review screenshots here (not at repo root).
- Do **not** ignore this folder in `.gitignore`; files here are intentional review evidence.
