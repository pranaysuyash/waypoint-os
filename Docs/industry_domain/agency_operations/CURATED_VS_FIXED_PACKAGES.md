# Domain Knowledge: Curated vs. Fixed Packages

**Category**: Agency Ecosystem  
**Focus**: Tailor-made itineraries vs. mass-market pre-defined packages.

---

## 1. Curated (Tailor-Made / Bespoke)
- **Logic**: The agent builds a "White Paper" itinerary from scratch based on the traveler's specific DNA.
- **Supply Chain**: The agent contacts multiple DMCs, hotels, and flight desks to "Stitch" the trip together.
- **Pricing**: High margin (often "Net" rates with a significant markup) or "Service Fee" based.
- **SOP**: Requires a **"Discovery Call"** to understand the traveler's psychological profile.

---

## 2. Pre-Defined (Fixed / Wholesale) Packages
- **Logic**: Off-the-shelf trips (e.g., "7 Days in Bali with Flights").
- **Supply Chain**: Bought from a "Tour Operator" or "Consolidator." The agency is just the reseller.
- **Pricing**: Low margin (Commission only); price-sensitive.
- **SOP**: Focused on **"Fulfillment Efficiency"** (booking the package in 5 minutes).

---

## 3. The "Dynamic Packaging" Hybrid
- **Logic**: Using tech (e.g., GDS or Expedia API) to bundle a flight and hotel on the fly.
- **Advantage**: Bypasses the "Package Travel Directive" (sometimes) and offers better pricing than separate bookings.
- **SOP**: The agent must ensure the "Legal Protection" (bonding) still applies if sold as a bundle.

---

## 4. Charter-Based Packages
- **Logic**: Tour operators rent the *entire plane* and *entire hotel* to drive costs down.
- **Risk**: If the plane doesn't fill, the operator loses money. If the plane has a technical issue, there are no "Alternative Flights" (Sub-charter required).

---

## 5. Proposed Scenarios
- **The "Curated Fatigue"**: An agent spends 20 hours building a custom $50k trip, but the client takes the itinerary and books it themselves on Expedia. The agent must implement a "Retainer Fee" SOP.
- **The "Package Failure"**: A client books a fixed package to the Maldives, but the "Included Transfer" doesn't run on Fridays. The agent must solve the "Hidden Dependency."
- **Markup Crisis**: A client finds the "Direct Rate" for a hotel in a curated itinerary and discovers a 40% markup. The agent must explain the "Value of Curation."
- **Charter Cancellation**: A charter flight is cancelled. The operator has no "Interline" agreement. The agent must find a way to get 180 people home.
