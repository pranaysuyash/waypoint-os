# Additional Scenario 71: Multi-Nationality Visa Coordination

**Scenario**: A group of travelers with different passports needs a shared itinerary, and the system must coordinate visa requirements for each nationality.

---

## Situation

A group booking includes travelers from multiple countries with varying visa rules.
The agent must ensure every traveler meets entry requirements for all stops.

## What the system should do

- Identify visa requirements separately for each passport holder
- Highlight any travelers who need additional documents or visas
- Recommend itinerary changes if a route is impractical for one or more nationalities
- Communicate the complexity clearly without overwhelming the customer

## Why this matters

Multi-nationality groups are common for weddings, family travel, and events.
A single oversight can delay the entire group or require last-minute changes.

## Success criteria

- The system tracks visa needs by nationality
- It prevents any traveler from being blocked at entry
- The agent provides a clear plan for each passport holder
