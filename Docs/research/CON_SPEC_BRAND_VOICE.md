# Con Spec: Agentic 'Brand-Voice' Guardian (CON-REAL-017)

**Status**: Research/Draft
**Area**: Agency Branding & Communication Consistency

---

## 1. The Problem: "The Generic Agent Voice"
In a multi-tenant SaaS, there is a risk that the AI agent sounds the same for every agency. A "Luxury-Concierge" agency in Mayfair should not sound like a "High-Energy-Adventure" agency in Queenstown. If the "Brand-Voice" is inconsistent across channels (WhatsApp, email, voice), it erodes traveler trust and weakens the agency's brand identity.

## 2. The Solution: 'Brand-Consistency-Protocol' (BCP)

The BCP acts as the "Digital-Brand-Manager."

### Governance Actions:

1.  **Brand-DNA Definition**:
    *   **Action**: Defining the agency's "Brand-Attributes" (e.g., Tone: Professional, Warm, Adventurous; Language: British-English; Formatting: Minimalist).
2.  **Multi-Channel-Voice-Sync**:
    *   **Action**: Ensuring that every message generated by the agent—regardless of the channel—is passed through the "Brand-Filter" to align with the DNA.
3.  **Adaptive-Tone-Scaling**:
    *   **Action**: Adjusting the tone based on the "Interaction-Context" (e.g., being "Concise & Urgent" during a flight delay, but "Expansive & Inspiring" during trip planning), while staying within the brand boundaries.
4.  **Brand-Violation-Detection**:
    *   **Action**: Monitoring all agent interactions for "Brand-Drift" (e.g., using slang that doesn't fit a luxury brand) and autonomously correcting the generation parameters.

## 4. Key Logic Rules

- **Rule 1: The 'Agency-Ownership' Priority**: The agency's "Brand-DNA" ALWAYS overrides default agent communication styles.
- **Rule 2: Contextual-Empathy-Constraint**: The agent MUST NOT prioritize brand voice over "Traveler-Empathy" during a crisis (e.g., don't be "Refined-Minimalist" if a traveler is distressed about a lost passport; switch to "Action-Oriented-Support").
- **Rule 3: Iterative-Brand-Learning**: The agent SHOULD learn from the agency owner's manual edits to agent messages to further refine the "Brand-Voice" over time.

## 5. Success Metrics (Brand)

- **Brand-Consistency-Score**: Traveler feedback on the consistency of the agency's communication across channels.
- **Owner-Message-Edit-Rate**: Decrease in the number of times an agency owner has to manually edit agent-generated messages for tone.
- **Brand-Recall-Index**: % of travelers who correctly identify the agency's brand attributes based on agent interactions.
