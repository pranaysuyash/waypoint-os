The project is an AI-assisted operating system for boutique travel agencies. 

**Core Thesis:**
It is not a "trip planner" but a workflow compression tool for solo/small agencies. It optimizes for:
1. Client Fit (Traveler suitability, not just group-level planning)
2. Conversion Likelihood
3. Margin Quality (Commercial viability)
4. Operational Ease (Using preferred supplier networks)
5. Risk Reduction (Detecting wasted spend, mobility issues, and pacing errors)

**Key Differentiators:**
- **Preferred Inventory First**: Prioritizes agency tie-ups over open-market search.
- **Wasted-Spend Detection**: Analyzes if expensive activities are actually usable by all group members (e.g., toddler/elderly suitability).
- **B2B Wedge**: Acts as an agency copilot, not a consumer replacement.
- **Hybrid Validation**: Uses a free "intelligence layer" (Itinerary Audit) to capture market data and lead-gen for agencies.

**Operational Flow:**
Lead $\rightarrow$ Qualify $\rightarrow$ Route (Package vs Custom) $\rightarrow$ Source (Partner vs Network vs Market) $\rightarrow$ Personalize $\rightarrow$ Quote $\rightarrow$ Revise $\rightarrow$ Ops/Execution $\rightarrow$ Memory.
